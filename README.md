# Expense Tracker CLI
A terminal-based Expense Tracker application built using Python.

## Features
* Log expenses with amount, category, and remarks
* View all recorded expenses
* Filter expenses by date or category
* Monthly expense report with per-category breakdown
* Persistent storage using Python's pickle module

## Concepts Used
* Functions
* Lists and Dictionaries
* File Handling
* Data Persistence with Pickle
* Loops and Conditionals

## How to Run
```bash
python main.py
```

## Future Improvements
* Exception handling for invalid inputs
* Export reports to CSV or JSON
* Use JSON instead of pickle for storage
* Edit or delete existing expenses
